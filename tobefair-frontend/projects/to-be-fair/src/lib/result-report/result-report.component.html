2BFAIR FAIRness Evaluation Report



Overall Result

This part shows the FAIRness, a percentage grade indicating how close a digital object is to fully abiding by the FAIR Principles, of your data.

Result by metric

This part shows the results divided by the metrics of each dimension and how much was achieved in each one.

Detailed results

This part is divided into principles. Each principle is divided by metrics. Each metric is divided into tests. In each test, it explains the test result, the score obtained, the maturity of each metric, a recommendation and what the person loses if they do not pass this test.
